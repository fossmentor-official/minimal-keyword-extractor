## Installation
```bash
git clone <repo_url>
cd minimal-keyword-extractor
pip install -r requirements.txt
```

## Run
**CLI**
```bash
python extractor.py
```

**Streamlit UI**
```bash
streamlit run streamlit_app.py
```

**API**
```bash
uvicorn main_api:app --reload
```
